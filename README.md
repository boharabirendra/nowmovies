# Clone of https://nowmovies.netlify.app/

## Tech used

- React.js
- Material UI

### Running locally

```sh
 git clone https://github.com/boharabirendra/nowmovies.git

 cd nowmovies

 Create .env file and see the example from .env.example

 npm install

 npm run dev
```
